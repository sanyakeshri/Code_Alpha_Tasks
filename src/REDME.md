🎓 Student Grade Tracker (Java)
This is my first fully self-coded Java project, developed during my internship at CodeAlpha.
It is a simple and beginner-friendly console application that helps track student names and their marks.

This project represents the starting point of my Java development journey, where I learned how to apply Object-Oriented Programming (OOP) concepts in a real project.

🚀 Features
● Build a Java program to input and manage student grades.
● Calculate average, highest, and lowest scores.
● Use arrays or ArrayLists to store and manage data.
● Display a summary report of all students.
● Make the interface console-based or GUI-based as desired.

🧠 What I Learned
While building this project independently, I practiced:

Creating classes and objects
Using constructors and getter methods
Working with ArrayList
Handling user input with Scanner
Designing a simple menu-driven Java program
Writing clean, readable, and modular code

   git clone

3. Compile the files
   javac *.java

4. Run the program
   java Main
